# ERICK - iOS

This folder contains the iOS implementation of ERICK (Ergonomic Radial Inclusive Controller Keyboard).

## Setup

iOS development is currently in progress. When the Xcode project is created, it will be placed here.

### Requirements

- macOS with Xcode installed
- iOS 14.0 or later target
- Swift 5.0+

### Getting Started

1. Install Xcode from the Mac App Store
2. Open the Xcode project (to be created)
3. Build and run on simulator or device

## Project Structure

## Development

The iOS app will provide the same ergonomic keyboard functionality as the Android version, adapted for iOS using native UIKit/SwiftUI frameworks.
