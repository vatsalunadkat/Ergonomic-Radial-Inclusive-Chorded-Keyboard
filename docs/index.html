What is ERICK?

ERICK is a chorded keyboard that replaces dozens of tiny keys with two intuitive joystick dials. Swipe on both dials to combine directions into character "chords" — making every letter, number, and symbol equally easy to type.

Touch-based swipe input

Physical controller input

Hands-free typing

How It Works

Three simple movements, one character. That's it.

1

Left Dial

Swipe the left joystick in a direction to select a character group (e.g. A–E).

2

Right Dial

Swipe the right joystick to select the specific character within that group.

3

Output

Both directions combine into a chord — the character appears instantly in your text field.

Key Features

Built from the ground up for speed, accessibility, and privacy.

🎯

Dual-Joystick Input

Two radial dials combine to form character chords. Every character requires equal effort — no more reaching for tiny keys.

🎨

Multiple Layouts

Choose from Logical (A–Z), Efficiency (frequency-optimized), or create your own custom layout.

🌙

Light & Dark Mode

Follow your system preference automatically, or set light/dark mode manually in settings.

🎮

Controller Support

Plug in a physical gaming controller and use its analog sticks for text input — great for consoles and TVs.

♿

Accessibility First

Colorblind palettes, left-handed mode, dyslexia-friendly fonts, and large touch targets for motor accessibility.

🔒

Privacy Focused

Zero data collection, no internet permissions, fully offline. 100% open-source so you can verify every line.

🎯

Typing Practice

Built-in relaxing typing game with curated quotes to help you learn chord combinations at your own pace.

✏️

Custom Layouts

Design your own keyboard layout that fits the way you think and type. Save and switch anytime.

100% Open Source

ERICK is built in the open with Kotlin Multiplatform. Inspect the code, suggest features, or contribute directly.
